RWAlgebra

How to use in HTML file:

  • Include script in HTML file:
    • Source: src/dist/rwalgebra-min.js
  • Equation, Expression objects, and parse function are stored in the rwalgebra global variable
  • To access:
    • const {Expression, Equation} = rwalgebra
    • Can now use Expression & Equation objects like normal

Example: eval

Test Expression:


Evaluating {x_1: 1, x_2: 2}

Answer:


Code:

const {Expression} = rwalgebra;
let ex = new Expression('(x_1+1)-(x_2+2)');
document.getElementById('expression').innerHTML = ex.toString();
document.getElementById('eval').innerHTML = ex.eval({x_1:1, x_2:2});
